Un ordinateur de bord dédié

Hardware :
  • Raspberry Pi 5
    • 8Gb, 86.60 €
    • 16Gb 135.60 €
  • Carte SD (sert de disque dur), de 16Gb à 512Gb, de 12 à 45 € (sur Amazon)
  • Cable USB (USB-A vers USB-C), alimentation (5V), adaptateur 12V-5V (genre allume-cigare)
  • Écran, interface HDMI, idéalement en 12V ou 5V.
    Comme par exemple celui-ci, à 120 €.
  • Clavier & souris. Il y en a de toutes sortes et de toutes tailles, de wireless à USB-A...
    Comme ici.
    Attention: quel genre de clavier ? QUERTY ou AZERTY ?
    Note: Sur un bateau, un trackball est souvent plus adapté qu'une souris.
    Les claviers avec un touch-pad (comme ici) sont aussi une option à ne pas négliger.
    Voir quelques exemples ici.
  • Et on peut y ajouter nombre de périphériques, comme une caméra, des hauts-parleurs, etc.
Software (tous gratuits) :
  • Operating System:
    • Raspberry Pi OS, 32 ou 64 bits, selon le modèle de RPi5 choisi
    • Twister OS
    • Windows and DOS emulators:
      • dosbox, with a sudo apt install dosbox
      • wine, with a sudo apt install wine

  • Apache2, PHP, MySQL/MariaDB, phpMyAdmin
  • VNC (remote desktop access)
  • OpenCPN, QGIS, Inkscape, GIMP, LibreOffice, etc...
  • StarLinux
