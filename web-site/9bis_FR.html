    • Du code, en Java, pour se faire son Compas Solaire Viking 
      # run the class app.samples.VikingSunCompass                                        
$ java -jar VikingSunCompass.jar -help  

Usage is :
  java [-DdeltaT=67.53] app.samples.VikingSunCompass <name value>
  names can be:
    -lat  for latitude, a decimal, in decimal degrees (default is 38.0)
    -size for the frame size (width = height), an integer, in pixels (default is 750)
    -help for help (you're on it)
Example:
  java app.samples.VikingSunCompass -lat 37.5 -size 600

This will generate image files (png) in the current directory.

 The length of the pole (pivot) is at the bottom left of the generated images
 The direction of the rose is the elevated pole (north if your latitude is north, south if your latitude is south).
